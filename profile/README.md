# WokSpec

**We build tools, then open source them so you can build too.**

WokSpec is a small team building AI-powered products on Cloudflare — fast, free to deploy, no locked infrastructure. We open source the things that matter most: the tools, the extensions, the templates.

---

## Open Source Projects

| Project | Description |
|---------|-------------|
| [**Eral**](https://github.com/WokSpec/Eral) | Open-source AI layer — browser extension, embeddable widget, and Cloudflare Worker API. Persistent memory, page context, deep integrations. |
| [**Dilu**](https://github.com/WokSpec/Dilu) | The WokSpec launchpad. Pick a template, configure your brand, ship a real product — deployed on Cloudflare, owned by you. |
| [**WokTool**](https://github.com/WokSpec/WokTool) | 80+ free browser-based tools for developers and designers. No login, no upload, runs in your browser. |
| [**Chopsticks**](https://github.com/WokSpec/Chopsticks) | Production-grade open-source Discord bot — music, moderation, economy, AI commands, and an agent pool. |

---

## The Stack

Everything WokSpec ships runs on the same stack you can use for free:

- **Cloudflare Pages** — edge-deployed Next.js apps
- **Cloudflare Workers** — serverless API and AI routing
- **Cloudflare KV + D1** — storage without a database bill
- **Workers AI** — on-device inference, no API key required

---

## Philosophy

> Build like us. Deploy like us. Own what you build.

We do not believe in SaaS lock-in, upload-to-our-servers tooling, or dashboards that obscure your own code. Every WokSpec product is something you could fork, deploy yourself, and own completely.
